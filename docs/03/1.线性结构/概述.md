# 线性结构
## 特点
1. 存在唯一的一个被称做“第一个”的数据元素﹔
2. 存在唯一的一个被称做“最后一个”的数据元素﹔
3. 除第一个之外,集合中的每个数据元素均只有一个前驱﹔
4. 除最后一个之外,集合中每个数据元素均只有一个后继。

## 分类
1. 数组
2. 链表
3. 队列
4. 栈
